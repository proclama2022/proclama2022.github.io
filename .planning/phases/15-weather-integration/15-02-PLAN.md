# Plan 15-02: Smart Logic & Climate Advice

## Goal
Implement the business logic to interpret weather data into actionable plant care advice (Rain Delay, Heat Acceleration).

## Tasks
- [x] **Climate Logic Service**: Create `services/climateLogic.ts`.
    - `analyzeWeather(data: WeatherData): ClimateAdvice`
    - Rules:
        - **Rain Delay**: If `current.rain > 5mm` OR `daily.rain_sum[0] > 5mm` (today) OR `daily.rain_sum[1] > 5mm` (yesterday) -> Advice: "Rain detected. Outdoor plants may not need water."
        - **Heat Warning**: If `daily.maxTemp` > 30°C for next 3 days -> Advice: "Heatwave approaching. Check thirsty plants daily."
        - **Cold Warning**: If `current.temp < 5°C` -> Advice: "Freezing temperatures. Protect sensitive plants."
- [x] **Store Integration**: Update `weatherStore.ts` to compute and expose `climateAdvice`.
- [x] **UI Updates**:
    - Update `components/WeatherWidget.tsx` to display the advice.
    - Add a collapsible/expandable section to the widget for details.
- [x] **Localization**: Add advice strings to `en.json` and `it.json`.

## Technical Details
- `ClimateAdvice` interface:
    ```typescript
    interface ClimateAdvice {
      type: 'rain' | 'heat' | 'cold' | 'neutral';
      messageKey: string; // key for i18n
      action?: 'delay' | 'water_more' | 'protect';
    }
    ```
