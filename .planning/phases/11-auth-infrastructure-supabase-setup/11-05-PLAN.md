  <name>Task 1: Create migration service with progress tracking</name>
  <files>services/migrationService.ts</files>
  <action>
Create services/migrationService.ts:

1. Import dependencies:
   - getSupabaseClient from '@/lib/supabase'
   - AsyncStorage from '@react-native-async-storage/async-storage'
   - type SavedPlant from '@/types'

2. Define MigrationProgress interface:
   - total: number
   - completed: number
   - currentPlantName: string
   - isCancelled: boolean
   - failed: number

3. Define MigrationResult interface:
   - success: number
   - failed: number
   - cancelled: boolean

4. Implement migratePlantsToSupabase():
   - Parameters: plants (SavedPlant[]), onProgress callback, signal object { cancelled: boolean }
   - Returns: Promise<MigrationResult>
   - Algorithm:
     a. Get supabase client
     b. Initialize counters: success = 0, failed = 0
     c. Loop through plants with index i:
        - Check signal.cancelled, break if true
        - Call onProgress({ total: plants.length, completed: i, currentPlantName, isCancelled: false })
        - Upload plant data to Supabase 'plants' table (table created in Phase 12)
        - Upload photos to Supabase Storage (bucket created in Phase 12)
        - Upload watering history to Supabase 'watering_history' table
        - On success: increment success counter
        - On error: log error, increment failed counter
     d. Return { success, failed, cancelled: signal.cancelled }

5. Implement compression helper:
   - compressImage() uses expo-image-manipulator
   - Resize to max 1200px width/height
   - Compress to quality 0.7
   - Returns compressed URI

6. Implement uploadPhoto():
   - Compress image
   - Upload to Supabase Storage bucket 'plant-photos'
   - Path: user_id/plant_id/timestamp.jpg
   - Returns public URL

7. Implement setMigrationFlag():
   - Store in AsyncStorage: '@plantid_has_migrated'
   - Value: JSON string with { timestamp, plantCount }
   - Called after successful migration

8. Implement getMigrationFlag():
   - Read from AsyncStorage
   - Returns null or parsed object
   - Used to check if user already migrated

9. Implement clearMigrationFlag():
   - Delete from AsyncStorage
   - Allows retry migration

10. Add JSDoc comments for all functions

Note: Supabase tables don't exist yet (created in Phase 12). This service will be tested then. For now, implement with mock placeholder comments.

Reference 11-RESEARCH.md "Pattern 4: Cancellable Migration with Progress".
