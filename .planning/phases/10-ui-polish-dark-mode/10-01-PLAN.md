**components/useColorScheme.ts** — Replace the single-line re-export with a store-aware hook:

```typescript
import { useColorScheme as useSystemColorScheme } from 'react-native';
import { useSettingsStore } from '@/stores/settingsStore';

export function useColorScheme(): 'light' | 'dark' {
  const storedScheme = useSettingsStore((state) => state.colorScheme);
  const systemScheme = useSystemColorScheme() ?? 'light';
  if (storedScheme === 'system') return systemScheme;
  return storedScheme;
}
```

This single change automatically propagates through `useThemeColors` (which calls `useColorScheme`) and through `app/_layout.tsx` `RootLayoutNav` (which also imports from `@/components/useColorScheme`). No other files require modification.

Do NOT use `useColorScheme` directly from `react-native` anywhere in this file — that would bypass the override.
