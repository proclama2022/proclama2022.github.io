# Plan 12-03: Profile UI Components

**Goal:** Create reusable Avatar and ProfileStats components with theming support and placeholder handling.

## Tasks

### Task 1: Create Avatar component

Create `components/Avatar.tsx`:

**Props interface:**
```typescript
interface AvatarProps {
  uri?: string | null; // Avatar URL from Supabase Storage
  size?: number; // Default 80 (diameter in pixels)
  borderColor?: string; // Optional border color
  borderWidth?: number; // Default 0
  style?: ViewStyle; // Additional container styles
}

interface AvatarWithStatsProps extends AvatarProps {
  showStats?: boolean;
  stats?: {
    followers?: number;
    following?: number;
  };
}
```

**Component structure:**

**Base `Avatar` component:**
- Render `ThemedView` with circular border
- Conditionally render:
  - If `uri` exists: `Image` component with `source={{ uri }}`
  - If no `uri`: `Ionicons` `person-circle-outline` icon (themed color)
- Circular styling:
  - `width: size`, `height: size`
  - `borderRadius: size / 2`
  - `overflow: 'hidden'`
- Border: Apply `borderColor` and `borderWidth` if provided

**Theming:**
- Use `ThemedView` wrapper for background color
- Use `useColorScheme()` hook for icon color:
  - Light mode: `Colors.light.text`
  - Dark mode: `Colors.dark.text`
- Placeholder icon color: `colors.text` (from useColorScheme)

**Image optimization:**
- Use `FastImage` from `react-native-fast-image` if available (optional enhancement)
- Fallback to standard `Image` component
- Add `resizeMode: 'cover'` for proper cropping

**Loading state:**
- Show placeholder icon while image loads
- Add `onLoad` callback to hide placeholder

### Task 2: Create ProfileStats component

Create `components/ProfileStats.tsx`:

**Props interface:**
```typescript
interface ProfileStatsProps {
  stats: {
    plants_identified: number;
    followers_count: number;
    following_count: number;
    joined_date?: string; // ISO timestamp
  };
  onStatPress?: (statType: 'plants' | 'followers' | 'following') => void;
  style?: ViewStyle;
}
```

**Note:** The `onStatPress` callback exists but actual list views (followers/following lists) are deferred to a future phase per the CONTEXT.md decision. Tapping stats will show the callback is called, but the navigation to list views is NOT implemented in this phase.

**Component structure:**

**Grid layout:**
- 2x2 grid (3 stats + joined date)
- Use `View` with `flexDirection: 'row'`, `flexWrap: 'wrap'`
- Each stat: `width: '48%'` (2 columns with gap)

**Stat item structure:**
```
┌─────────────────────┐
│     🌱 42           │
│   Plants Identified │
└─────────────────────┘
```

**Stat items:**
1. **Plants Identified:**
   - Icon: `leaf-outline` (Ionicons)
   - Value: `stats.plants_identified`
   - Label: "Plants" (localized via i18n)

2. **Followers:**
   - Icon: `people-outline` (Ionicons)
   - Value: `stats.followers_count`
   - Label: "Followers" (localized)

3. **Following:**
   - Icon: `person-add-outline` (Ionicons)
   - Value: `stats.following_count`
   - Label: "Following" (localized)

4. **Joined Date:**
   - Icon: `calendar-outline` (Ionicons)
   - Value: Formatted date (e.g., "Joined Jan 2026")
   - Format: `MMM YYYY` from `joined_date`
   - No tap handler (not interactive)

**Interactivity:**
- If `onStatPress` provided:
  - Wrap stat items (excluding joined date) in `TouchableOpacity`
  - Add opacity change on press (`activeOpacity: 0.7`)
  - Call `onStatPress(statType)` with stat type
- If no `onStatPress`: Render as static `View`

**Theming:**
- Use `ThemedView` wrapper
- Icon color: `colors.tint` (from useColorScheme)
- Value color: `colors.text` (primary text)
- Label color: `colors.tabIconDefault` (secondary text)
- Background: `colors.background` (from ThemedView)

**Number formatting:**
- Format large numbers: "1.2K", "10.5K" for > 1000
- Use `Intl.NumberFormat` for localization
```typescript
const formatNumber = (num: number): string => {
  if (num >= 1000) {
    return `${(num / 1000).toFixed(1)}K`;
  }
  return num.toString();
};
```

### Task 4: Add date formatting utility

Create `lib/utils/dateFormatter.ts`:

**Export function:**
```typescript
formatJoinedDate(isoDate: string, locale: string = 'en'): string
```

**Algorithm:**
1. Parse ISO date string to Date object
2. Format as "MMM YYYY" (e.g., "Jan 2026")
3. Localize month name based on `locale` parameter
4. Return formatted string with "Joined " prefix

**Example:**
```typescript
formatJoinedDate('2026-03-02T10:00:00Z', 'en') // "Joined Mar 2026"
formatJoinedDate('2026-03-02T10:00:00Z', 'it') // "Joined mar 2026"
```
