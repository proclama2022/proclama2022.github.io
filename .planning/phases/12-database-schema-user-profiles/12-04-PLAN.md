**Algorithm for viewing own profile:**
1. Get `user` from `useAuthStore()`
2. Get `currentProfile`, `isLoading`, `error` from `useProfileStore()`
3. If no user, show "Sign in to view profile" message
4. If user but no profile, call `fetchCurrentProfile(user.id)`
5. Show loading spinner while `isLoading`
6. Show error message if `error` exists
7. Render profile layout:
   - **Avatar:** Large avatar (120px) with `Avatar` component
   - **Display name:** Bold text, large font size
   - **Bio:** Regular text, multiline if needed
   - **Edit button:** Show only if viewing own profile
   - **Stats grid:** `ProfileStats` component with stats from `currentProfile.stats`

**Theming:**
- Use `ThemedView` for background
- Use `ThemedText` for all text
- Colors from `useColorScheme()` hook

**Refresh on focus:**
Use `useFocusEffect` from `@react-navigation/native`:
```typescript
useFocusEffect(
  useCallback(() => {
    if (user && currentProfile) {
      useProfileStore().refreshStats();
    }
  }, [user?.id])
);
```

**Verification:** Explicitly verify that when user switches away from Profile tab and returns, stats are refreshed by calling `refreshStats()`.

**Note:** Viewing other users' profiles is deferred to Phase 13 (navigation from feed). Set `isOtherUser` state for future use.

**Avatar upload flow:**
1. User taps "Change Avatar" button
2. Show action sheet: "Choose from Library" | "Take Photo" | "Cancel"
3. On selection, call `profileStore.uploadAvatar(source)`
4. Show loading spinner while uploading
5. On success, update `avatarUri` state
6. On error, show alert with error message

**Form fields:**
- **Display Name:**
  - `TextInput` with `maxLength={50}`
  - Show character count: `displayName.length}/50`
  - Required field (validation error if empty on save)

- **Bio:**
  - `TextInput` with `multiline={true}`, `numberOfLines={3}`
  - `maxLength={500}`
  - Show character count: `bio.length}/500`
  - Optional field

**Save button logic:**
1. Validate display name (not empty)
2. Call `profileStore.updateProfile({ display_name: displayName, bio })`
3. Show loading spinner
4. On success: Call `onClose()`
5. On error: Show alert with error message

**Cancel button:**
- Discards unsaved changes
- Calls `onClose()`

**Theming:**
- Modal background: `colors.surface`
- Input backgrounds: `colors.background`
- Text colors: `colors.text`, `colors.tabIconDefault`
- Primary button: `colors.tint`
