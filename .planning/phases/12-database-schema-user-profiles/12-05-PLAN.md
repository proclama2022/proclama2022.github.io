# Plan 12-05: Public Profile Viewing & Follow System

**Goal:** Enable viewing other users' profiles via push navigation and follow/unfollow functionality.

## Tasks

### Task 1: Create public profile screen

Create `app/profile/[userId].tsx` (Expo Router dynamic route):

**URL pattern:** `/profile/{userId}`

**Screen structure (same as profile tab but with follow button):**
```
┌─────────────────────────────┐
│        [Avatar 120px]       │
│      Display Name           │
│      Bio text               │
│                             │
│    [Follow / Following]     │  <-- NEW
│                             │
│  ┌──────┐ ┌──────┐         │
│  │ 🌱 42│ │ 👥 12│         │
│  │Plant │ │Follow│         │
│  └──────┘ └──────┘         │
│  └──────┘ └──────┘         │
└─────────────────────────────┘
```

**Component props:**
- No props (data from route params)

**Component state:**
```typescript
const { userId } = useLocalSearchParams<{ userId: string }>();
const [profile, setProfile] = useState<ProfileWithStats | null>(null);
const [isFollowing, setIsFollowing] = useState(false);
const [isLoading, setIsLoading] = useState(true);
const currentUser = useAuthStore().user;
```

**Algorithm:**
1. Get `userId` from route params
2. Check if `userId === currentUser?.id`:
   - If yes, redirect to `/tabs/profile` (own profile)
   - If no, continue
3. Fetch profile data:
   - Call `profileService.fetchProfile(userId)`
   - Call `followService.isFollowing(currentUser.id, userId)`
4. Set state with results
5. Show loading spinner while fetching
6. Show error message if fetch fails
7. Render profile layout:
   - Same as profile tab (Avatar, display name, bio, stats)
   - Replace "Edit Profile" button with FollowButton component
   - Pass `isFollowing` state to FollowButton

**Header:**
- Set title to profile's display name
- Add back button (default from stack navigation)

**Theming:**
- Use same theme as profile tab
- Use `ThemedView`, `ThemedText`

### Task 2: Create Follow service

Create `services/followService.ts`:

**Type definitions:**
```typescript
import { Follow } from '@/types/profile';

interface FollowServiceResult {
  success: boolean;
  error?: string;
}
```

**Export functions:**

**`followUser(followerId: string, followingId: string): Promise<FollowServiceResult>`**
Algorithm:
1. Validate: `followerId !== followingId` (can't follow self)
2. Call `followUser(followerId, followingId)` from `lib/supabase/profiles.ts`
3. On success: Return `{ success: true }`
4. On error: Return `{ success: false, error: error.message }`

**`unfollowUser(followerId: string, followingId: string): Promise<FollowServiceResult>`**
Algorithm:
1. Call `unfollowUser(followerId, followingId)` from `lib/supabase/profiles.ts`
2. On success: Return `{ success: true }`
3. On error: Return `{ success: false, error: error.message }`

**`checkIsFollowing(followerId: string, followingId: string): Promise<boolean>`**
Algorithm:
1. Call `isFollowing(followerId, followingId)` from `lib/supabase/profiles.ts`
2. Return boolean result

### Task 3: Create FollowButton component

Create `components/FollowButton.tsx`:

**Props interface:**
```typescript

  isFollowing: boolean;
  onFollow: () => Promise<void>;
  onUnfollow: () => Promise<void>;
  disabled?: boolean;
  style?: ViewStyle;
}
```

**Component state:**
```typescript
const [isLoading, setIsLoading] = useState(false);

```

**Button appearance:**
- **Not following:**
  - Background: `colors.tint` (primary color)
  - Text: "Follow" (white)
  - Border: None
  - Icon: None

- **Following:**
  - Background: Transparent
  - Text: "Following" (`colors.tint`)
  - Border: `1px solid colors.tint`
  - Icon: `checkmark-circle` (Ionicons) on left

**Button size:**
- Width: 120px (or fit content)
- Height: 40px
- Border radius: 20 (pill shape)
- Center alignment

**Interaction:**
1. User taps button
2. Set `isLoading: true`
3. If `currentIsFollowing` is true:
   - Call `onUnfollow()`
   - On success: Set `currentIsFollowing: false`
   - On error: Show alert
4. If `currentIsFollowing` is false:
   - Call `onFollow()`
   - On success: Set `currentIsFollowing: true`
   - On error: Show alert
5. Finally: Set `isLoading: false`

**Loading state:**
- Show `ActivityIndicator` (small) when `isLoading`
- Disable button while loading

**Theming:**
- Use colors from `useColorScheme()`

### Task 4: Wire follow functionality in public profile screen

Update `app/profile/[userId].tsx`:

**Add follow handlers:**
```typescript
const handleFollow = async () => {
  const result = await followService.followUser(currentUser.id, userId);
  if (result.success) {
    setIsFollowing(true);
    // Refresh stats to update follower count
    const updatedProfile = await profileService.fetchProfile(userId);
    if (updatedProfile.success) {
      setProfile(updatedProfile.data);
    }
  } else {
    Alert.alert('Error', result.error);
  }
};

const handleUnfollow = async () => {
  const result = await followService.unfollowUser(currentUser.id, userId);
  if (result.success) {
    setIsFollowing(false);
    // Refresh stats
    const updatedProfile = await profileService.fetchProfile(userId);
    if (updatedProfile.success) {
      setProfile(updatedProfile.data);
    }
  } else {
    Alert.alert('Error', result.error);
  }
};
```

**Render FollowButton:**
```typescript
<FollowButton
  isFollowing={isFollowing}
  onFollow={handleFollow}
  onUnfollow={handleUnfollow}
/>
```
