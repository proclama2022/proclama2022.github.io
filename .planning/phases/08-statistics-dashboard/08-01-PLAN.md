In computeStats(), after the existing weeklyData loop (lines 52-66), add a parallel loop for completed reminders:

```typescript
// Weekly reminder data (last 7 days — completed reminders only, grouped by reminder.date)
const weeklyRemindersData: number[] = [];
for (let i = 6; i >= 0; i--) {
  const d = new Date(today);
  d.setDate(d.getDate() - i);
  const dateStr = d.toISOString().split('T')[0];
  let count = 0;
  plants.forEach(p => {
    if (p.reminders) {
      p.reminders.forEach(r => {
        if (r.completed && new Date(r.date).toISOString().split('T')[0] === dateStr) {
          count++;
        }
      });
    }
  });
  weeklyRemindersData.push(count);
}
```

Add `weeklyRemindersData` to the return object alongside weeklyData.

IMPORTANT: Group by `reminder.date` (scheduled date), NOT by a completion date — the Reminder type has no completedDate field.
