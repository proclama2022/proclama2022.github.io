  <name>Create BannerAdWrapper component with Pro gating</name>

  <action>
Create components/BannerAdWrapper.tsx following Pattern 3 from RESEARCH.md:

1. Import dependencies:
   import React from 'react';
   import { View, StyleSheet, Platform } from 'react-native';
   import { BannerAd, BannerAdSize, TestIds } from 'react-native-google-mobile-ads';
   import { useSafeAreaInsets } from 'react-native-safe-area-context';
   import { useProStore } from '@/stores/proStore';

2. Define ad unit IDs with Platform.select and __DEV__ check:
   const AD_UNIT_ID = Platform.select({
     ios: __DEV__ ? TestIds.BANNER : process.env.EXPO_PUBLIC_ADMOB_IOS_BANNER_ID,
     android: __DEV__ ? TestIds.BANNER : process.env.EXPO_PUBLIC_ADMOB_ANDROID_BANNER_ID,
   });

3. Create BannerAdWrapper component:
   - Get isPro from useProStore()
   - Get insets from useSafeAreaInsets()
   - Return null early if isPro is true (don't render anything for Pro users)
   - Return View with position: 'absolute', bottom: 0, left: 0, right: 0
   - Add paddingBottom: insets.bottom for safe area
   - Render BannerAd with:
     * unitId={AD_UNIT_ID}
     * size={BannerAdSize.BANNER}
     * requestOptions={{ requestNonPersonalizedAdsOnly: true }}
     * onAdFailedToLoad: console.log (hide cleanly, no placeholder)

4. Create StyleSheet:
   - container: absolute positioning, white background
   - Center the ad horizontally

5. Export component as default

DO NOT show any placeholder or error message when ad fails to load — just hide cleanly.
  </action>
  <verify>grep -E "export.*BannerAdWrapper|useProStore|BannerAd" components/BannerAdWrapper.tsx returns matches</verify>
  <done>BannerAdWrapper created with Pro check, safe area padding, clean error fallback</done>
