<task type="auto">
  <name>Create RevenueCat purchase service wrapper</name>
  <files>services/purchaseService.ts</files>
  <action>
Create services/purchaseService.ts following Pattern 2 from RESEARCH.md:

1. Import dependencies:
   import { Purchases } from 'react-native-purchases';
   import { useProStore } from '@/stores/proStore';
   import { PurchaseError } from '@/types';

2. Define constants:
   const ENTITLEMENT_ID = 'no_ads';
   const OFFERING_ID = 'default'; // RevenueCat default offering

3. Implement initPurchases():
   - Check if Purchases is configured (avoid double-init)
   - Call Purchases.configure({ apiKey: process.env.REVENUECAT_API_KEY })
   - Catch and log errors

4. Implement checkProStatus(): Promise<boolean>
   - Try: const customerInfo = await Purchases.getCustomerInfo()
   - Check: const isPro = customerInfo.entitlements.active[ENTITLEMENT_ID] !== undefined
   - Update store: useProStore.getState().setIsPro(isPro)
   - Update timestamp: useProStore.getState().setLastVerified(new Date().toISOString())
   - Catch: return useProStore.getState().isPro (fallback to cached)
   - Return: boolean

5. Implement purchasePro(): Promise<{ success: boolean; error?: PurchaseError }>
   - Get offerings: const offerings = await Purchases.getOfferings()
   - Get product: const product = offerings.current?.availableProducts[0]
   - Throw error if !product
   - Purchase: const { customerInfo } = await Purchases.purchaseProduct(product.identifier)
   - Check entitlement: const isPro = customerInfo.entitlements.active[ENTITLEMENT_ID] !== undefined
   - Update store if successful
   - Handle userCancelled error gracefully
   - Return { success: isPro } or { success: false, error: {...} }

6. Implement restorePurchases(): Promise<boolean>
   - Call: const customerInfo = await Purchases.restorePurchases()
   - Check entitlement
   - Update store if found
   - Return boolean

All functions must handle errors gracefully and never crash the app.
  </action>
  <verify>grep -E "export.*initPurchases|export.*checkProStatus|export.*purchasePro|export.*restorePurchases" services/purchaseService.ts returns 4 matches</verify>
