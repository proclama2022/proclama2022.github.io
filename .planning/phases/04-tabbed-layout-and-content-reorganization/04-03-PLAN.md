**Reusable CareSection component (internal to CareTab.tsx):**
```typescript
function CareSection({ label, hasData, children }: {
  label: string;
  hasData: boolean;
  children: React.ReactNode;
}) {
  return (
    <View style={styles.section}>
      <Text style={styles.sectionLabel}>{label}</Text>
      {hasData
        ? children
        : <Text style={styles.notAvailable}>{t('detail.care.notAvailable')}</Text>
      }
    </View>
  );
}
```

**Section implementations:**

1. **Watering section** (always has data if care exists):
   - `hasData={!!care}`
   - Show: watering frequency in days, existing CareInfo watering badge pattern
   - If no care: show notAvailable

2. **Light section** (always has data if care exists):
   - Show sunlight value mapped to human text (full-sun → "Full Sun" / "Pieno Sole", etc.)
   - `hasData={!!care}`

3. **Temperature section:**
   - Show basic tempMin–tempMax range row: `care.tempMin–care.tempMax°C`
   - If `care.seasonalTemps` exists, also render seasonal list:
     ```
     Spring: 15–20°C
     Summer: 20–28°C
     ```
     Use `t('detail.care.spring')` etc. for season labels, `t('detail.care.tempRange', { min, max })` for the range.
   - `hasData={!!care}`

4. **Fertilization section:**
   - `hasData={!!(care?.fertilization)}`
   - If data: show schedule and type using `isItalian ? care.fertilization.schedule.it : care.fertilization.schedule.en`

5. **Pruning section:**
   - `hasData={!!(care?.pruning)}`
   - If data: show when and how texts using the bilingual pattern

6. **Pests section (expandable):**
   - `hasData={!!(care?.pests && care.pests.length > 0)}`
   - Use `useState<number | null>(null)` for `expandedPestIndex`
   - For each pest, render a `TouchableOpacity` row showing:
     - `isItalian ? pest.name.it : pest.name.en` (bold)
     - `isItalian ? pest.description.it : pest.description.en` (muted)
     - When expanded: `isItalian ? pest.remedy.it : pest.remedy.en` with a muted "Remedy:" label above
     - Show `t('detail.care.tapToExpand')` hint text below description when collapsed
   - `onPress={() => setExpandedPestIndex(expandedPestIndex === index ? null : index)}`

**Layout:** Outer ScrollView (flex:1), padding: 16. Each section has a bottom border separator except the last.

**Handle no care at all:** If `!care`, render a single centered message: "Care information not available for this species" — but still render the section headings per locked decisions (CARE-05).

**Styles:** Match existing app color palette (#2e7d32 green accents, #1a1a1a text, #888 muted, #f5f5f5 background). Section labels: uppercase, fontSize 11, fontWeight '600', color '#888', letterSpacing 0.4.
