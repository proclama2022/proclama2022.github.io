### Task 2: Leaderboard Service

<task type="auto">
  <name>Task 2: Add getLeagueLeaderboard</name>
  <files>services/leagueService.ts</files>
  <action>
    Add to leagueService.ts:

    **getLeagueLeaderboard(userId: string, limit = 30): Promise<LeaderboardEntry[]>**

    Query logic:
    1. Get user's current cohort membership
    2. Join league_memberships with user_progress and profiles
    3. Calculate xp_this_week = xp_total - xp_at_start for each member
    4. Rank by xp_this_week DESC
    5. Flag is_promotion_zone (rank <= 10)
    6. Flag is_relegation_zone (rank >= 26 for 30-user cohort)
    7. Flag is_current_user

    Return top 30 entries with user info (avatar, display_name, level)

    Handle edge case: user not in any cohort → return empty array
  </action>
  <verify>
    <automated>grep -q "getLeagueLeaderboard" services/leagueService.ts</automated>
  </verify>
  <done>getLeagueLeaderboard returns ranked entries with zone flags</done>
</task>
